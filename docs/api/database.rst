oscarpes.database
====================

Query interface to an oscarpes v3 Lance + Zarr database.

:class:`OSCARDatabase` provides SQL-style metadata filtering, batch loading
for ML, CD-ARPES partner discovery, PyTorch streaming, and pool statistics —
all without loading full ARPES arrays unless explicitly requested.
